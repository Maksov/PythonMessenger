# README #


### Project description ###

A simple Python messenger for learning sockets, threads, PyQT5, SQLAlchemy, etc...

### Install ###

```pip install aemessenger```
or
```pip3 install aemessenger```

### Usage ###

Server: ```python3 -m aemessenger.server```
Qt Client: ```python3 -m aemessenger.server```

#### Sample logins ####
```Dany , testlogin```
